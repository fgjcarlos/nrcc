🚀 nrcc

Node-RED Control Center

Single-binary management UI for Node-RED with authentication, backups, logs, and more.

Quick Install

1. Download & Install

curl -fsSL https://get.nrcc.dev/install.sh | sh

2. Configure System Service

sudo nrcc install

3. Open in Browser

Visit http://localhost:3001 and set up your admin user

Options

Pin a specific version

NRCC_VERSION=v1.0.0 curl -fsSL https://get.nrcc.dev/install.sh | sh

Custom install directory

INSTALL_DIR=$HOME/.local/bin curl -fsSL https://get.nrcc.dev/install.sh | sh

GitHub Pages not available? Use the raw script from GitHub:
